1)Co to są pseudoklasyk?

Co to są pseudoklasyk? CSS przewiduje kilka ściśle określonych klas pozwalających na formatowanie niektórych elementów dokumentu są to Pseudoklasy.

2)Uwagi dotyczące dwukropka :

Uwaga dotycząca dwukropka: Szalenie istotnym elementem w tej konstrukcji jest dwukropek występujący pomiędzy selektorem a nazwą klasy.

3)Tabela(wykonana w htmlu) opisującą pseudoklasy.

Sposoby definiowania kolorów w CSS

Sposoby definiowania kolorów w CSS

Sposób Przykład Opis
Nazwa koloru red, blue, green Kolor określany za pomocą nazwy, np. red, blue, green.
RGB rgb(255, 0, 0) Określenie koloru za pomocą trzech składowych: czerwony, zielony, niebieski. Wartości od 0 do 255.
RGBA rgba(255, 0, 0, 0.5) Podobne do RGB, ale dodaje przezroczystość (alpha) w zakresie 0 (przezroczysty) do 1 (nieprzezroczysty).
HEX #FF0000 Kolor określany za pomocą szesnastkowego kodu, np. #FF0000 dla czerwonego.
HSL hsl(0, 100%, 50%) Kolor określany za pomocą skali barw (Hue), nasycenia (Saturation) i jasności (Lightness).
HSLA hsla(0, 100%, 50%, 0.5) Podobnie jak HSL, ale dodaje przezroczystość (alpha).
CMYK cmyk(0%, 100%, 100%, 0%) Określanie koloru na podstawie modeli używanych w druku (Cyan, Magenta, Yellow, Black).
