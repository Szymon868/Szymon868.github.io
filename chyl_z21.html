Identyfikatory — ID

Pojęcie Opis
Co znaczy skrót ID? ID oznacza identyfikator unikalnego elementu w dokumencie HTML.
Jaki jest cel stosowania ID? ID pozwala na jednoznaczne identyfikowanie i stylizowanie elementu.
Jakie jest znaczenie znaku #? Znak # oznacza identyfikator w CSS, np. #nazwa_id.
Jak następuje wywołanie ID? W HTML: id="nazwa_id". W CSS: #nazwa_id { właściwości }
Uwagi na temat stosowania ID
  • ID musi być unikalne w obrębie jednej strony.
  • Błędem jest wielokrotne użycie tego samego ID, ale nie wpływa to na działanie strony.
  • Jeśli element może wystąpić wiele razy, lepiej używać klasy (class) zamiast ID.
