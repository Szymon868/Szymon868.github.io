Sposoby podłączania CSS do kodu strony WWW

1. Inline CSS

Stylizacja dodawana bezpośrednio w atrybucie elementu HTML.
2. Internal CSS

Stylizacja dodawana w sekcji <style> w kodzie HTML.
3. External CSS

Stylizacja ładowana z zewnętrznego pliku CSS.
4. CSS w Import

Stylizacja importowana za pomocą reguły @import.
5. Stylizacja za pomocą Frameworka

Stylizacja z użyciem gotowych bibliotek CSS, np. Bootstrap.
